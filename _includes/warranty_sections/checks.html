Before Submitting a Claim

Most issues can be resolved quickly with a few simple checks. These are the same steps our support team will ask about:

  • Ensure all cables are fully seated and correctly matched (especially ARGB 5V vs RGB 12V)
  • Test the product on a different header or device if possible
  • Check your motherboard BIOS settings for fan and ARGB control
  • Try a different SATA or power cable if using a hub
  • Confirm that controller buttons / software haven’t overridden your preferred settings

If these steps don’t solve it — don’t worry. We’ll pick it up from there.
