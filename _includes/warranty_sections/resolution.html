6. Quick Checks (Before Contacting Support)

RGB / Lighting Issues

  • Ensure ARGB is connected to a 5V 3-pin header (not 12V RGB)
  • Check alignment arrows on connectors
  • Try a different ARGB port on the hub
  • Disable motherboard LED overrides if applicable

Fan Not Spinning

  • Ensure the PWM cable is fully inserted
  • Check BIOS is set to PWM mode instead of DC
  • Try a different hub port
  • Test with a known-working cable

Hub Not Responding

  • Confirm SATA power is connected correctly
  • Ensure ARGB and PWM leads are attached
  • Check for hub LED indicators

If these checks don’t resolve the issue, we’ll take it from there.
