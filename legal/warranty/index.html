6. Quick Checks (Before Contacting Support)

RGB / Lighting Issues

  • Ensure ARGB is connected to a 5V 3-pin header (NOT 12V RGB)
  • Check the alignment arrows on the connectors
  • Try a different ARGB port on the hub
  • Disable motherboard LED overrides if present

Fan Not Spinning

  • Ensure the PWM cable is fully inserted
  • Check BIOS settings are set to PWM mode, not DC
  • Swap the fan to a different hub port
  • Try a known-working cable

Hub Not Responding

  • Check SATA power is connected properly
  • Ensure both ARGB and PWM leads are connected
  • Check for hub LED power indicators
